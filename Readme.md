## Create a vertual enviornment
```python
python -m venv venv
```
---

## Run vertual enviornment
```python
./venv/scripts/activate
```
---

## Stop vertual enviornment
```python
deactivate
```
---

## Installation All Modules

```bash
pip install -r requirements.txt
```
---


## To Run Fast Api Server
```python
uvicorn main:app --reload
```
